# dotfiles/victoru

managed via [stow](https://www.gnu.org/software/stow/stow.html)

### chunkwm & skhd
requires:
- victoru/chunkwm
- victoru/skhd

```shell
cd dotfiles
stow --dir=victoru --target=$HOME chunkwm
stow --dir=victoru --target=$HOME skhd
```

### vim
```shell
cd dotfiles
stow --dir=victoru --target=$HOME vim
```

### git

```shell
cd dotfiles
stow --dir=victoru --target=$HOME git
```
